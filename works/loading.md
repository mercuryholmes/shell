
## 説明
プログレスバーを表示するコマンドです。

* bash
```bash
I=$(printf "Loading[%10s]"); echo "$I"; for i in {1..10}; do I=${I/ /■}; echo "$I"; sleep 1; done; echo "Load complate";
```

* Perl
```Perl
perl -e 'print my $txt = "Loading[".(" " x 10)."]\n"; for(1..10){ $txt =~ s/\s/■/; print $txt; sleep 1; } print "Load complate\n"; ';
```
